#  Fire-Water-Grass: The Ultimate RPS

A thematic twist on the classic Rock-Paper-Scissors game! **Fire-Water-Grass** brings the elemental triangle of Pokémon-style combat into a simple, fun, and fast-paced digital game.

---

##  How it Works
The rules are simple but the stakes are high:
*  **Fire** burns **Grass**
*  **Grass** absorbs **Water**
*  **Water** extinguishes **Fire**

---

##  Features
* **Instant Result Logic:** Fast calculation of winner/loser/draw.
* **Themed UI:** Visuals designed around the three core elements.
* **Score Tracking:** Keep track of your wins and losses against the computer.
* **Responsive Design:** Playable on both desktop and mobile devices.

---

##  Getting Started

### Prerequisites
To run this project locally, you just need a modern web browser (or the appropriate runtime if you are using Node.js/Python).

### Installation & Usage
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/duckguy1208/Fire-Water-Grass-RPS.git](https://github.com/duckguy1208/Fire-Water-Grass-RPS.git)
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd Fire-Water-Grass-RPS
    ```
3.  **Launch the game:**
    https://fire-water-grass.streamlit.app/
